游戏方案

## 游戏名称（暂定）：桌面旅者
> 一款以悬浮窗常驻桌面的古代行商 Roguelike 放置游戏

## 游戏概述：
这是一款基于 **Unity URP** 开发的桌面悬浮窗游戏。它融合了 **放置挂机、虚实结合与 Roguelike 随机性**。玩家将扮演一位古代行商，通过现实世界的键盘敲击、鼠标移动与在线时长，驱动商队在广袤的古代大陆上旅行。游戏以 **透视 X 轴 2D 像素剪影** 风格呈现，如同一张活着的古卷铺在桌面之上。

---

| 要素 | 实现描述 |
|:---|:---|
| **1. 挂机** | 随时间自动累积「旅费」（基础收益）。 |
| **2. 放置** | 配置随从、装载货物、学习能力，构建成长 Build。 |
| **3. 虚实结合** | 键盘/鼠标操作实时转化为「脚程」与「校准」，破除挂机资源壁。 |
| **4. 旅行** | 沿随机生成的节点地图前进，探索古城、荒野、集市。 |
| **5. 事件** | 上线时触发积压的遭遇战、奇遇或交易，决策影响深远。 |
| **6. Roguelike** | 随机地图、随机事件、角色特质与能力组合，重开性强。 |
| **7. 角色框架** | 玩家与 NPC 共用：职业、金币、资产、特质、能力。 |
| **8. 引擎与管线** | Unity 引擎 + URP 渲染管线，C# 脚本驱动。 |
| **9. 前台悬浮** | 无边框、置顶、可穿透点击的悬浮窗，如输入法般安静陪伴。 |
| **10. 镜头风格** | 透视 X 轴 2D，多层视差滚动，像素剪影美术。 |
