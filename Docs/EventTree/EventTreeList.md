# 事件树列表 (EventTreeList)

## 概述

本文件记录游戏中所有事件树的配置。事件树定义了复杂的多分支叙事结构，控制玩家在关键剧情节点的选择和后续发展。

## 事件树结构说明

| 字段 | 说明 |
|------|------|
| 节点数量 | 事件树中的总节点数（包含根节点和叶节点） |
| 分支数 | 主要分支路径的数量 |

## 新手教程事件树 (NewPlayerGuide)

### Core.EventTree.NewPlayerGuide

| 属性 | 值 |
|------|-----|
| ID | Core.EventTree.NewPlayerGuide |
| 名称 | 新手教程事件树 |
| 描述 | 引导新玩家了解游戏基础玩法，包括移动、战斗、收集等系统 |
| 节点数量 | 8 |
| 分支数 | 2 |

**叙事主线：**
玩家在村口醒来，遇到指引者N对基。通过与N对基的对话，学习基本的操作和概念。事件树包含两个主要分支：
- 战斗倾向路线：重点学习战斗系统，获得武器推荐
- 探索倾向路线：重点学习收集和探索，获得道具推荐

最终两个分支都会汇合，触发序章完结事件，解锁完整游戏内容。

## 森林探险事件树 (ForestAdventure)

### Core.EventTree.ForestAdventure

| 属性 | 值 |
|------|-----|
| ID | Core.EventTree.ForestAdventure |
| 名称 | 森林探险事件树 |
| 描述 | 玩家进入古老的黑森林，探索遗迹、遭遇神秘生物、发现隐藏的秘密 |
| 节点数量 | 12 |
| 分支数 | 4 |

**叙事主线：**
玩家接受村长的委托，进入黑森林寻找失踪的探险队。事件树包含四条主要分支路径：
- 探险队路线：直接寻找探险队，遭遇怪物巢穴
- 遗迹路线：发现古代遗迹，获得历史线索
- 精灵路线：遇到森林精灵，获得精灵帮助
- 黑暗路线：遭遇黑暗势力，触发隐藏剧情

每个分支都有独特的事件链，最终会聚合成一个共同结局。

## 宝藏猎手事件树 (TreasureHunt)

### Core.EventTree.TreasureHunt

| 属性 | 值 |
|------|-----|
| ID | Core.EventTree.TreasureHunt |
| 名称 | 宝藏猎手事件树 |
| 描述 | 玩家获得一张古老地图，开始寻找传说中埋藏在世界各地的宝藏 |
| 节点数量 | 15 |
| 分支数 | 5 |

**叙事主线：**
玩家在旧书店获得一张神秘的地图碎片，上面标记着传说中「黄金城」的位置。事件树包含五条主要分支路径：
- 沙漠线索：前往炎热沙漠，遭遇沙虫和古墓
- 雪山线索：攀登寒冷雪山，发现冰封古城
- 丛林线索：穿越密林，遭遇部落和毒虫
- 海岛线索：乘船寻找海底遗迹，面对海怪
- 深渊线索：进入地下世界，发现远古文明

每条路径都能获得一块地图碎片，收集全部碎片后解锁最终宝藏位置。

## 英雄传承事件树 (HeroLegacy)

### Core.EventTree.HeroLegacy

| 属性 | 值 |
|------|-----|
| ID | Core.EventTree.HeroLegacy |
| 名称 | 英雄传承事件树 |
| 描述 | 传说中的英雄在临终前将使命传承给玩家，开启一段拯救世界的旅程 |
| 节点数量 | 20 |
| 分支数 | 6 |

**叙事主线：**
这是游戏的核心主线事件树，讲述了英雄协会的起源和与黑暗帝王的千年之战。事件树包含六条主要分支路径：
- 英雄试炼：接受英雄协会的考验，证明自己的资格
- 黑暗觉醒：调查黑暗势力的复苏迹象
- 远古同盟：寻找失落的远古种族结盟
- 神器收集：收集散落各地的神器碎片
- 王城陷落：见证王城被黑暗攻陷
- 最终决战：对抗黑暗帝王，决定世界命运

每个分支都会影响最终的结局走向，共有三种不同结局。

## 黑暗秘密事件树 (DarkSecret)

### Core.EventTree.DarkSecret

| 属性 | 值 |
|------|-----|
| ID | Core.EventTree.DarkSecret |
| 名称 | 黑暗秘密事件树 |
| 描述 | 玩家逐渐发现这个王国的建立背后隐藏着不可告人的黑暗历史 |
| 节点数量 | 18 |
| 分支数 | 5 |

**叙事主线：**
这是一个暗黑风格的支线事件树，揭示王国建立的真实历史。事件树包含五条主要分支路径：
- 王宫密道：发现王宫地下室的秘密档案
- 教廷阴谋：揭露光明教廷与黑暗势力的关联
- 皇室血统：发现皇室血统的惊天秘密
- 亡魂安息：帮助无法安息的亡魂完成遗愿
- 真相抉择：面对真相做出最终选择

这个事件树与英雄传承事件树有交叉点，真相会影响最终决战的结局。
