## 四、阻塞问题

### 4.1 csproj问题

```
问题: Assembly-CSharp.csproj缺少引用
├── CombatModule.cs 未包含
├── GamePlay/ 目录未包含
└── 导致107个编译错误
```

**解决方案**:
1. 在Unity Editor中打开项目（自动重新生成）
2. 或删除 `Assembly-CSharp.csproj` 让Unity重新生成
