2. **敌人闪避属性**: 原来使用playerDodgeChance*0.5f，修复为独立的dodgeChance字段
3. **百分比减伤**: 扁平减伤改为 defense/(defense+100) 递减百分比减伤
4. **单敌战斗硬编码**: 0.05f硬编码改为变量引用
5. **默认初始化**: CreateEnemyFromTemplate和敌人初始化添加dodgeChance=0.05f
