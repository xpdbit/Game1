## 一、代码质量审查

### 1.1 命名规范

| 检查项 | 要求 | 优先级 |
|--------|------|--------|
| 命名空间 | `Game1.{模块}` | 必须 |
| 类名 | PascalCase | 必须 |
| 方法名 | PascalCase | 必须 |
| 私有字段 | `_camelCase` 或 `m_CamelCase` | 必须 |
| 常量 | 全大写+下划线 | 必须 |
| 接口名 | `I` 前缀 | 必须 |
| 枚举值 | PascalCase | 必须 |

### 1.2 代码风格

```csharp
// ✅ 正确示例
namespace Game1.Modules.Combat
{
    public class CombatSystem : ICombatModule
    {
        private const float CRIT_CHANCE_BASE = 0.1f;
        private float _currentDamage;

        public void ExecuteAttack(CombatantData attacker, CombatantData defender)
        {
            if (attacker == null || defender == null)
                throw new ArgumentNullException(nameof(attacker));

            _currentDamage = CalculateDamage(attacker, defender);
        }

        private float CalculateDamage(CombatantData attacker, CombatantData defender)
        {
            return attacker.attack * (1f - defender.defense / 100f);
        }
    }
}

// ❌ 错误示例
namespace combat
{
    public class combatsystem
    {
        float damage;
        public void attack(float atk, float def) {
            damage = atk * (1-def/100);
        }
    }
}
```

## 三、性能审查

### 3.1 Unity Profiler关注点

| 问题 | 影响 | 解决方案 |
|------|------|----------|
| 每帧分配 | GC压力 | 对象池 |
| GetComponent in Update | CPU开销 | Awake/Start缓存 |
| FindObjectsOfType | 性能灾难 | 单例或ServiceLocator |
| 同步加载 | 主线程阻塞 | Addressables异步 |
| SendMessage | 性能差 | 事件总线 |

### 3.2 代码级优化

```csharp
// ✅ 正确 - 缓存组件引用
public class MyBehaviour : MonoBehaviour
{
    private Image _backgroundImage;
    private Text _statusText;

    private void Awake()
    {
        _backgroundImage = GetComponent<Image>();
        _statusText = transform.Find("Status").GetComponent<Text>();
    }

    private void Update()
    {
        // 使用缓存的引用，不调用GetComponent
        _statusText.text = DateTime.Now.ToString();
    }
}

// ❌ 错误 - 每帧调用GetComponent
private void Update()
{
    GetComponent<Image>().color = Color.red; // 每帧分配
}
```

### 3.3 循环优化

```csharp
// ✅ 正确 - 循环外创建
var targetPosition = CalculateTargetPosition();
for (int i = 0; i < 1000; i++)
{
    enemies[i].MoveTo(targetPosition);
}

// ❌ 错误 - 循环内创建
for (int i = 0; i < 1000; i++)
{
    var pos = CalculateTargetPosition(); // 每帧调用1000次
    enemies[i].MoveTo(pos);
}
```

### 4.2 线程安全

```csharp
// ✅ 正确 - Unity对象线程安全
public class MyManager : MonoBehaviour
{
    private readonly object _lock = new object();
    private int _score;

    // 主线程调用
    public void AddScore(int value)
    {
        lock (_lock)
        {
            _score += value;
        }
    }

    // 主线程调用
    public int GetScore()
    {
        lock (_lock)
        {
            return _score;
        }
    }
}
```
